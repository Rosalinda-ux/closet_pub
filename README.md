# closet manager

洋服の管理アプリです。自分の所有している洋服を登録して、把握ができます。

# 機能一覧

洋服の新規登録

登録した洋服の一覧の閲覧(画像と名前)

登録した洋服の詳細情報の確認・編集・削除
